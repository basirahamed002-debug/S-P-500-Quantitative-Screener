# 📊 S&P 500 Quantitative Screener
## 🚀 Top 20 Buy Signals · Advanced Analysis

## 📌 Project Overview

This project is an advanced quantitative equity research dashboard that identifies the Top 20 Buy Signals from the S&P 500 using a multi-factor scoring model.
The dashboard analyzes 1.89M+ rows of historical stock data (2010–2024) across 502 companies to uncover high-performing investment opportunities.
It combines risk, return, and momentum indicators to generate a composite score for stock selection.

## 🎯 Objective
### The goal of this project is to:

Build a data-driven stock screening system
Identify high-performing stocks using quantitative metrics
Visualize insights using an interactive dashboard
Demonstrate data analytics + financial analysis skills

## 📊 Key Features
## 🔹 Multi-Factor Scoring Model
Stocks are ranked using a weighted scoring system:
  📈 Sharpe Ratio → 40% weight
  💰 1-Year Return → 35% weight
  ⚡ 30-Day Momentum → 25% weight
